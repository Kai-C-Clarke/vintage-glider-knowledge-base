# How to Contribute

## Easiest way — Google Drive

Click this link and drop your files straight in:

👉 **https://drive.google.com/drive/folders/1TkleSGRi_L-Vv60ijjaLGmXRwfzECZcc**

You can upload PDFs, JPEGs, TIFFs, Word documents, scanned images — anything. If you have a folder of related files, you can upload the whole folder at once.

## File naming tips (not essential, but helpful)

If you can, name files with the type first:
- `Olympia2b_MaintenanceManual_1954.pdf`
- `Ka6_RiggingNotes_handwritten.jpg`
- `SHK1_G-DCKT_repair_1987.pdf`

But honestly — don't let naming put you off. Any file is better than no file.
